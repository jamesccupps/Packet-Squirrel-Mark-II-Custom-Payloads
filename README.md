# Packet-Squirrel-Mark-II-Custom-Payloads
Custom payloads I made for the Packet Squirrel, will be updating and adding more.

Packet Capture Manual - Starts in bridge mode, has web ui enabled to allow remote access. Push Button to start capture and button to stop capture. Can be repeated without restarting device and saves seperate files for each run.

Packet Capture Auto - Starts in bridge mode, has web ui enabled. Auto starts capture, stops with button or when drive is full. Saves IP to txt file, drive can be removed, file extracted and drive replaced, then allowing direct connection without scanning a network.
